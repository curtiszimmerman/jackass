# jackass

Jackass: The Package.

## Usage

```js
var jackass = require('jackass');

console.log( jackass(0) ); // 0
console.log( jackass(1) ); // 1

// special feature: also works with non-numeric input
console.log( jackass('s') ); // s
```

## Installation

With [npm](http://npmjs.org) do:

```bash
$ npm install jackass
```
